
claude code有五种扩展方式，分别是subAgent、SlashCommand、hook、skill、mcp和一种统一的分发方式plugin，用对这些扩展方式就可以吧claude code打造成最适合研发团队的助手。让团队的效率拉满
# Claude Code API 使用指南
我们把研发流程中的知识、经验和工具用正确的扩展能力封装起来
1、Slash command斜杠命令，封装的是研发流程中的动作比如需求分析，架构设计
2、subAgent子代理，封装的是研发流程中的角色比如架构师，业务专家
3、hook钩子，回调的标准动作，比如审计日志、飞书通知
4、skill技能，封装的是工作流，比如项目部署、自动化测试
5、mcp，封装的是外部接口，比如langgraph官网、公司知识库等。
这样一套下来可以让团队成员以最低的接入成本用到最强最统一的研发助手
6、plugin插件，统一的分发方式，用于将不同的扩展能力组合起来，形成一个完整的研发助手
